# Gemstone Rewards V1 — Clean Install

Default admin login: `admin` / `ChangeMe123!`

## Start fresh

1. Create a new Google Apps Script project.
2. Paste the included `Code.gs`.
3. Run `setup()` once.
4. Deploy as a Web App: Execute as Me, access Anyone.
5. Copy the `/exec` URL into `config.js`.
6. Upload `index.html`, `admin.html`, `styles.css`, `app.js`, `admin.js`, and `config.js` to a new GitHub repository.
7. Enable GitHub Pages from the `main` branch and root folder.
